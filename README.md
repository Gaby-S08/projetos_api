# 📌 Projeto: Integração com APIs (ViaCEP, IBGE e GitHub)

## 📖 Descrição

Este projeto contém três exemplos práticos de integração com APIs utilizando **HTML, CSS (Bootstrap)** e **JavaScript**. O objetivo é demonstrar como consumir dados de serviços externos e exibi-los de forma dinâmica na interface.

Os exemplos abordam:

* Busca de endereço pelo CEP (ViaCEP)
* Listagem de estados e cidades (IBGE)
* Criação de um crachá digital com dados do GitHub

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons
* JavaScript (Fetch API)

## 🧩 Funcionalidades

### 🔎 1. Consulta de CEP (ViaCEP)

Arquivo: `ex01_viaCep.html` 

* Formulário moderno e centralizado
* Busca automática de endereço a partir do CEP
* Preenchimento dinâmico dos campos
* Interface estilizada com Bootstrap

---

### 🌎 2. Consulta IBGE

Arquivo: `ex02_ibge.html` 

* Consumo da API do IBGE
* Listagem de estados brasileiros
* Possibilidade de carregar cidades por estado
* Interface simples com layout centralizado

---

### 🪪 3. Crachá Digital (GitHub)

Arquivo: `ex03_cracha_digital.html` 

* Busca de usuário do GitHub
* Exibição de:

  * Foto de perfil
  * Nome
  * Bio
* Criação de um “crachá digital” estilizado
